# Security Policy

Cloud2 Labs takes security seriously within the scope of reference
implementations.

## Reporting Vulnerabilities

If you discover a security issue, please report it responsibly by contacting
Cloud2 Labs directly. Do not disclose vulnerabilities publicly before they
are reviewed.

## Scope Limitations

This repository is not designed for production deployment and does not
guarantee protection against security threats.

Users are responsible for implementing appropriate security controls,
hardening, monitoring, and compliance measures before any real-world use.

Cloud2 Labs does not provide security guarantees, audits, or certifications
for this software.
